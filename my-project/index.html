API개발 | 웹페이지 개발 | 유지보수

_어드민페이지 통합 및 포팅 (Nimda 2차 :도매-주문)

  • 쿼리 성능 개선 작업
  • 코드 리펙토링 및 코드 모듈화
  • 기능 추가 및 개선 작업

_사입사 솔루션 개발

  • 비즈니스 요구사항 파악 및 DB모델링 (유저, 팀, 도매주문)
  • Restful API 서버 개발
  • 클라이언트 - 회원가입 페이지, 어드민페이지 - (팀/주문관리) 구현
